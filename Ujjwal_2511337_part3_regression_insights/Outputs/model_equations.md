# Model Equations

## Dummy Variable Approach

Two categorical variables were converted into dummy variables:

### Store Type

Reference category: **Residential**

Dummy variables created:

* High_Street
* Mall
* Airport

Residential was used as the reference category to avoid the dummy variable trap.

### Region

Reference category: **West**

Dummy variables created:

* North
* South
* East

West was used as the reference category to avoid redundancy.
